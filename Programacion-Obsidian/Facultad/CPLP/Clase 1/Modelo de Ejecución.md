Define cómo se ejecuta un programa. 

| Modelo | Descripción |
| ------ | ----------- |
| **[[Compilación\|Compilado]]** | Traducido a código máquina antes de ejecutarse |
| **[[Interpretación\|Interpretado]]** | Ejecutado por un intérprete en tiempo de ejecución |
| **[[Traducción Híbrida\|Híbrido]]** | Bytecode + máquina virtual (ej. [[JVM]]) |

> Ver también: [[Lenguaje de Programación]], [[Compilación]], [[Interpretación]]
